## How much fiber bot
Бот, который поможет подсчитать количество потребляемой клетчатки. 

В настоящий момент он ограничен в выборе продуктов, но со временем будет развиваться и расширять свои возможности.

### Что он умеет сейчас
1. Получить информацию о боте
2. Заполнить профиль пользователя, сохранив `username` пользователя, а также желаемую суточную норму клетчатки пользователя.
3. Подсчитать количество клетчатки в продукте, который ввел пользователь.
4. Выдать данные по клетчатке за сутки.
